Project title:

Enhancing User Experience in Mobile Banking Apps

Project Objectives:

  1. To gain insights into the current mobile banking habits and preferences of potential users.
  2. To identify challenges and areas for improvement in existing mobile banking apps.
  3. To determine the essential features and security measures users value in a mobile banking app.
  4. To gather feedback on additional functionalities users would like to see in a mobile banking app.
  5. To explore user opinions on trust, security, and potential enhancements in mobile banking apps.

Project Timeline:

Days 1-2: Development of interview questions and survey questions
Days 3-4: Interview data collection
Days 5-6: Survey data collection
Days 7-8: Data analysis and report writing
Day 9: Presentation and sharing of findings
Day 10: Project conclusion and final report preparation

Budget:

The project does not require a budget as it primarily relies on the team's skills and existing resources.
